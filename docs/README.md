# 李宏毅深度强化学习笔记(LeeDeepRL-Notes)
李宏毅老师的《深度强化学习》是强化学习领域经典的中文视频之一。李老师幽默风趣的上课风格让晦涩难懂的强化学习理论变得轻松易懂，他会通过很多有趣的例子来讲解强化学习理论。比如老师经常会用玩 Atari 游戏的例子来讲解强化学习算法。对于想入门强化学习又想看中文讲解的人来说绝对是非常推荐的。
